```
ip.src == 192.168.2.50 && ip.dst == 192.168.20.205 && tcp.dstport == 80 && tcp.flags == 0x02 &&
tcp.seq == 0 을 차단합니다./탐지합니다./허용합니다.

access-list 100
```
