# 로그인 admin / toor로 설정
<img width="1040" height="1040" alt="image" src="https://github.com/user-attachments/assets/388a94c6-ea54-48b0-aa29-f19bde9acf53" />

2번 부터 진행
```
1. 시스템 정보 확인

get system status


2. 네트워크 장치 IP 주소 설정

config system interface
    edit "port1"
        set mode static
        set ip 192.168.2.100 255.255.255.0
        set allowaccess http https ping ssh
    next
    edit "port2"
        set mode static
        set ip 192.168.20.100 255.255.255.0
        set allowaccess http https ping ssh
    next
end

get system interface physical
show system interface port1
show system interface port2


3. DNS 서버 지정

config system dns
    set primary 8.8.8.8
end

get system dns


4. 기본 게이트웨이 지정

config router static
    edit 1
        set gateway 192.168.2.254
        set device "port1"
    next
end

get router info routing-table all


5. 인터넷으로 Ping 테스트

execute ping 8.8.8.8
execute ping www.google.com
execute ping update.fortiguard.net


[참고] NAT 설정(보류)

config firewall policy
    edit 1
        set name "NAT_Policy"
        set srcintf "port2"
        set dstintf "port1"
        set action accept
        set srcaddr "all"
        set dstaddr "all"
        set schedule "always"
        set service "ALL"
        set nat enable
    next
end


[참고] FortiCloud Support Portal에서 라이센스 삭제 (권장)

1. Fortinet Support Portal에 접속하여 기존 계정으로 로그인합니다.

2. 상단 메뉴에서 Asset Management (또는 Products -> Product List)를 클릭합니다.

3. 목록에서 현재 사용 중인 FortiGate VM의 시리얼 번호(S/N)를 찾습니다.

4. 무료 버전의 경우 보통 FGVMEV로 시작합니다.

5. 해당 자산을 클릭하여 상세 페이지로 들어간 뒤, 관리 메뉴에서 Decommission 또는 Unregister 관련 옵션이 있는지 확인합니다.

주의: 무료 평가판(Permanent Trial)은 포털에서 직접 삭제가 안 되는 경우가 많습니다. 
이럴 때는 '기존 VM' 자체에서 연결을 끊어야 합니다.

# FortiCloud 연결 상태 해제 시도
config system central-management
    set status disable
end

# 라이선스 정보 초기화 (주의: 설정이 날아갈 수 있음)
execute factoryreset
```
