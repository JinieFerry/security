# 1. HTTP 메서드

GET과 POST 두 가지는 꼭 알아야 하는 메서드. 

URL에서의 + 기호는 공백을 의미함. 

Referer:목적지를 도착하기 전에 경유를 하고 왔다. 경유지 = 어디를 통해서 왔는지가 뜸.

쿠키는 과자처럼 접속한 흔적을 말함. 쿠키정보는 노출 되면 안됨. 다른 사람이 나인 것 처럼 연결 될 수 있음.
